## css 
- floats, display, box model도 중요하지만 flex box를 알면 굉장히 유연한 ui 만들기 가능
- SASS나 PostCSS같은걸로 duplication이 많은 요소들을 간편하게 관리 가능
- css framework : Bootstrap 예전에 많이썼지만, react 나오면서 잘 안쓰는 추세
- css architecture - BEM 아티클 몇개 읽어보기
